# E-Commerce ETL Pipeline (SQL)

## Overview

This project demonstrates an end-to-end ETL pipeline for an e-commerce dataset using SQL and MySQL. It focuses on transforming raw data into structured tables for efficient querying.

## Project Workflow

1. Create database tables
2. Load raw data
3. Transform and clean data
4. Perform analysis using SQL queries

## Files

* 01_create_tables.sql → Table creation (DDL)
* 02_load_data.sql → Data loading
* 03_transformations.sql → Data cleaning & transformations
* 04_analysis.sql → Analytical queries

## Tech Stack

* SQL
* MySQL

## How to Run

1. Run 01_create_tables.sql
2. Run 02_load_data.sql
3. Run 03_transformations.sql
4. Run 04_analysis.sql
